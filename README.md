## Описание проекта

3D платформер-шутер, созданный на Unity 2022.1.19f1. Игра представляет собой аркадный шутер с элементами выживания, где игрок сражается с различными врагами, используя разнообразное оружие и собирая лут для выживания.

## Основные возможности

### Игровая механика

- **Управление игроком**
  - Горизонтальное движение (A/D или стрелки)
  - Прыжки (Space)
  - Приседание (Ctrl/S) - уменьшает коллайдер для прохождения под препятствиями
  - Физика с трением и ограничением максимальной скорости

- **Система здоровья**
  - Максимальное здоровье: 8 единиц
  - Визуальное отображение здоровья через UI иконки
  - Система неуязвимости после получения урона (1 секунда)
  - Возможность восстановления здоровья через лут

- **Система оружия**
  - **Пистолет** - базовое оружие с неограниченным боезапасом
  - **Автомат** - автоматическое оружие с ограниченным количеством патронов
  - **Jump Gun** - специальное оружие, позволяющее делать рывок назад при выстреле (заряжается 3 секунды)
  - Переключение между оружиями
  - Визуальные эффекты выстрела (вспышка, частицы)

- **Система врагов**
  - **Курица (Hen)** - преследует игрока, нанося урон при контакте
  - **Медведь, Свинья, Кролик, Белка** - различные типы врагов
  - **Ракета (Rocket)** - самонаводящийся снаряд, преследующий игрока
  - **Морковь (Carrot)** и  **Жёлудь (Acorn)** используется врагами как снаряд
  
  - Система здоровья врагов с событиями при получении урона и смерти

- **Система лута**
  
  - **Патроны (Loot Bullets)** - пополняет боезапас для автомата
  - Здоровье  (Loot Health) - пополняет запас здоровья главного героя
  

- **Специальные механики**
  - **Замедление времени** - при зажатии правой кнопки мыши время замедляется
  - Система активации объектов по расстоянию (оптимизация производительности)
  - Звуковое сопровождение (выстрелы, получение урона, подбор лута)

## Структура проекта

### Основные папки

- **Assets/Scripts/** - основной код игры
  - `PlayerMove.cs` - управление движением игрока
  - `PlayerHealth.cs` - система здоровья игрока
  - `PlayerArmory.cs` - управление арсеналом оружия
  - `Guns/` - скрипты оружия (Gun, Pistol, Automat, JumpGun)
  - `Enemy Base/` - базовая система врагов
  - `HealthUI.cs` - отображение здоровья в UI
  - `SoundManager.cs` - управление звуками
  - `TimeManager.cs` - управление замедлением времени

- **Assets/Prefabs/** - префабы игровых объектов
  - Враги: Bear, Hen, Pig, Rabbit, Squirrel
  - Лут: Carrot, Loot_Bullets, Loot_Heal
  - Эффекты: DieEffect, hit - yellow 2
  - Снаряды: Bullet, Rocket, Acorn

- **Assets/Scenes/** - игровые сцены
  - `SampleScene.unity` - основная игровая сцена

- **Assets/Models/** - 3D модели
- **Assets/Materials/** - материалы для объектов
- **Assets/Audio/** - звуковые эффекты
- **Assets/Effects/** - визуальные эффекты

## Технические детали

### Используемые технологии

- **Unity 2022.1.19f1** - игровой движок
- **C#** - язык программирования
- **Unity Physics** - физический движок для коллизий и движения
- **TextMeshPro** - система текста для UI
- **Unity Visual Scripting** - визуальное программирование (частично)

### Архитектура кода

- **Объектно-ориентированное программирование** - использование наследования (Gun → Pistol, Automat)
- **Unity Events** - система событий для связи компонентов
- **Component-based architecture** - модульная архитектура Unity

## Управление

- **WASD / Стрелки** - движение
- **Space** - прыжок
- **Ctrl / S** - приседание
- **Левая кнопка мыши** - стрельба
- **Правая кнопка мыши** - замедление времени
- **Left Shift** - использование Jump Gun (когда заряжен)

## Особенности реализации

1. **Физика движения** - реалистичная физика с трением и ограничением скорости
2. **Система урона** - гибкая система нанесения урона через коллизии и триггеры
3. **Оптимизация** - активация объектов по расстоянию для улучшения производительности
4. **UI система** - динамическое отображение здоровья и боезапаса
5. **Звуковая система** - управление звуками и музыкой

## Требования

- Unity 2022.1.19f1 или выше
- Windows/Mac/Linux
- Минимальные системные требования Unity 2022.1

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте проект в Unity 2022.1.19f1
3. Откройте сцену `Assets/Scenes/SampleScene.unity`
4. Нажмите Play для запуска игры
